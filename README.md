# wechatGroupRobot
微信群机器人，在微信群上每天早晨根据天气提醒大家穿衣或带伞，
每隔一段时间提醒喝水，结合百度AI的API做一点geek的事！添加语音识别功能，添加了图片识别功能,这是a分支新添加的功能,哈哈哈
